# Sales Performance Analysis Dashboard using Power BI and SQL
Learning data visualization using Power BI and SQL. First attempt in creating a full dashboard on PowerBI.

## Project Overview
This project explores sales performance trends using SQL and Power BI.
The Superstore dataset from Kaggle was analyzed to identify revenue drivers, customer behavior, product performance, and potential business growth opportunities.

The goal of the project was to practice:
* SQL data analysis
* Business-oriented questioning
* Dashboard design in Power BI
* Data storytelling and visualization

## Business questions
1. Which products, regions, and customer segments contribute the most to overall revenue, and where is the business overly dependent?
2. Which customer segments generate the highest value, and how can the company better target and retain these segments?
3. Which product sub-categories are underperforming, and what actions should be taken to improve or optimize their performance?
4. Which regions or states show the greatest growth potential beyond current high-performing areas?

## Key insights
* The West region generated the highest sales revenue across all regions.
* Corporate customers contributed significantly to overall revenue.
* Phones and Storage were the top 2 performing product sub-categories.
* Office Supplies recorded the highest order volume.
